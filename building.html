Building instructions

FEEngine is written in C#, and requires version 9.0. Therefore, it can only be built on Windows, as of writing this document. (June 3, 2021)
There is also a native implementation, but it is obsolete in favor of the C# version.
Requirements:

  • Visual Studio 2019
  • Mono (x86 or x64, depending on which architecture you wish to build for)
Once you have pulled the source, run "windows-genprojects.bat" to generate build files. Then build the C# and C++ projects like normal.
